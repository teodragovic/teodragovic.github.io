The trick is still solid. In fact, I decided to add it to this blog!

```scss
.post:after {
    content: "";
    position: fixed;
    width: 100%;
    height: 80px;
    left: 0;
    bottom: 0;
    // Allow text behind fade to be selectable
    pointer-events: none;
    // Make sure actual color matches page background
    background: linear-gradient(transparent, white);
}
```
